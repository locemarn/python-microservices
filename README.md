# python-microservices

## Steps
1. Create a Python Virtual Environment `python3 -m venv ~/.venv` or `virtualenv ~/.venv`
2. Create an empty files:
- touch requirements.txt
- touch Dockerfile
- touch Makefile
- mkdir mylib
- touch mylib/__init__.py
- touch mylib/logic.py
- touch main.py
3. Populate Makefile
